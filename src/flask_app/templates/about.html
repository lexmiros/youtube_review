A simple project created to improve my programming and data engineering skills. Extracts the YouTube JSON data via the Google API.
Calls the API through reqeusts instead of the Python wrapper created by Google. Uses webscraping to gather the YouTube user's
channel ID via their username. Uses Pandas to transform the data, formatting, cleaning, and creating new data from obtained data.
The data is finally loaded into the Flask webapp to create graphs.

Tech Stack:


  • Python
    • Pandas
    • Flask
    • BeautifulSoup
  • JavaScript
    • ChartJS
  • HTML
  • CSS
